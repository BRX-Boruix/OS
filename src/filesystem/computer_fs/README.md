# Computer:/ 文件系统实现

这是 Boruix OS 的 `computer:/` 虚拟文件系统实现，将硬件抽象为文件系统接口。

## 目录结构

```
computer_fs/
├── README.md           # 本文件
├── vfs_core/          # 虚拟文件系统核心
├── device_layer/      # 设备抽象层
├── state_files/       # 状态文件管理
├── directory_mgmt/    # 目录管理
├── tools/             # 开发工具
└── tests/             # 测试代码
```

## 开发阶段

### 第一阶段：基础框架 (当前)
- 虚拟文件系统核心
- 设备抽象层
- 状态文件支持
- 基础目录结构
- 开发工具和测试
